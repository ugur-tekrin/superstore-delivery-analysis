# superstore-delivery-analysis
Delivery performance analysis using Tableau and the Superstore dataset.

## Tableau Workbook
The Tableau packaged workbook (.twbx) is included in this repository.
You can download and open it in Tableau Desktop to explore the dashboard.

## Dataset
Sample Superstore Dataset
~10,000 orders
Period: 2020–2023

## Tools
Tableau Public
Data Visualization
Calculated Fields
Geographic Analysis

## Analysis Tasks
Sales and Profit time series analysis
Profitability by category and shipping mode
Delivery time distribution analysis
Average delivery time by U.S states
Interactive dashboard creation

## Dashboard
Tableau Public Dashboard
https://tinyurl.com/mufzzwzy

## Key Insights
Profitability varies across product sub-categories.
Certain shipping modes show lower profitability for specific products.
Most deliveries occur within a few days.
Delivery performance differs across U.S states.
